# Zappts Corporate Theme Example

This example demonstrates all the slide classes and features available in the Zappts corporate theme.

## Usage

```yaml
---
marp: true
theme: zappts-dark-theme
class: proposal
---
```

## Key Features Demonstrated

### 1. Proposal Cover Slide (`proposal`)
- Professional proposal information layout
- Client branding area
- Corporate styling

### 2. Section Dividers (`section-title`)
- Full-width section breaks
- High visual impact
- Clean typography

### 3. Content Slides (`title-content`)
- Structured content presentation
- Professional spacing
- Brand-consistent styling

### 4. Single Column (`onecol`)
- Full-width text layout
- Emphasis paragraph support
- Good readability

### 5. Two Column (`twocol`)
- Side-by-side comparisons
- Responsive grid layout
- Balanced content distribution

### 6. Topic Slides (`main-topic`)
- Large announcement format
- Subtitle support
- Minimal, impactful design

### 7. Title with Support Box (`title-sub-right`)
- Main content on left
- Supporting information on right
- Professional business layout

### 8. Title with Content Block (`title-sub-right-block`)
- Text content on left
- Image/media on right
- Visual emphasis

## Brand Colors Used

- **Primary Blue:** #4285F4 - Headers and accents
- **Orange Accent:** #FFAB40 - Highlights and CTAs  
- **Teal:** #0097A7 - Secondary elements
- **Emphasis Gray:** #646B6B - Special emphasis text
- **Dark Background:** #1E1E1E - Slide backgrounds
- **Light Text:** #E8E8E8 - Body text

## Typography Features

- **Montserrat font family** for professional appearance
- **Hierarchical heading system** with distinct colors
- **Emphasis styling** for important information
- **Consistent spacing** throughout all slide types

This example showcases the complete range of features available in the Zappts corporate theme for creating professional business presentations.
